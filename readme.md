# OhMyFood 

Projet 4 du parcours "Intégrateur Web" chez OpenClassrooms. 
L'objectif est de développer le site OhMyFood en "Mobile First" et de le dynamiser avec des animations CSS en utilisant le préprocesseur Sass.
